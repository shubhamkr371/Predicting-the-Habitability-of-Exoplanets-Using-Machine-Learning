# 🌌 Predicting the Habitability of Exoplanets Using Machine Learning

Welcome to the Exoplanet Habitability Predictor! This project leverages machine learning to assess the potential habitability of exoplanets based on their observed features. Explore the cosmos with data-driven insights and interactive visualizations.

---

## 🚀 Project Overview
This project aims to:
- Analyze exoplanet data
- Preprocess and rank planets by habitability
- Train ML models to predict habitability
- Provide a web interface for exploration

---

## 🧩 Features
- Data preprocessing and cleaning
- Multiple ML models for habitability prediction
- Interactive web frontend (HTML, JS, CSS)
- RESTful backend (Flask)
- Visualizations and habitability ranking

---

## 📁 Project Structure
```
backend/      # Flask API and utilities
frontend/     # Web UI (HTML, JS, CSS)
data/         # Processed datasets
models/       # Saved ML models
notebooks/    # Research and experiments
ML_Model_Training_Code.ipynb  # Main notebook
```

## ⚙️ Setup Instructions
1. **Clone the repository:**
	```bash
	git clone https://github.com/shubhamkr371/Predicting-the-Habitability-of-Exoplanets-Using-Machine-Learning.git
	cd Predicting-the-Habitability-of-Exoplanets-Using-Machine-Learning
	```
2. **Install dependencies:**
	- Backend: `cd backend && pip install -r requirements.txt`
	- Frontend: Open `frontend/index.html` in your browser
3. **Run the backend server:**
	```bash
	cd backend
	python app.py
	```
4. **Explore the frontend:**
	- Visit `http://localhost:5000` or open the HTML file directly
